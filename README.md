# Svelte Tetris

This is a hobby project. This code implement the game Tetris using SvelteKit and TailwindCSS.
![cap1](docs/capture_1.jpeg?raw=true)
![cap2](docs/capture_2.jpeg?raw=true)

## Developing

Install packages before first run:

```bash
pnpm install
```

```bash
pnpm run dev
```

## Building

Requires an adapter.

```bash
pnpm build
```
